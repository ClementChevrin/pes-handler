TYPE TNatDec
Type Description
Nature de la décision. Les termes "autorisation budgétaire" et "reports de crédits" sont utilisés pour les instructions M21 et M22.
Seules les valeurs 01 Budget primitif, 02 Décision Modificative, 03 Budget Supplémentaire sont autorisées et 10 CFU.
String Enumération avec les valeurs suivantes :
  • 01 : Budget primitif, (autorisation budgétaire).
  • 02 : Décision modificative.
  • 03 : Budget supplémentaire.
  • 09 : Compte administratif.
  • 10 : CFU.

TYPE TNatFonc
Type Description
String Enumération avec les valeurs suivantes :
  • 1 : Le budget est voté par nature.
  • 2 : Le budget est voté par fonction.
  • 3 : Le vote du budget est mixte.

TYPE TNatVote
Type Description
Niveau de vote du budget : indique si le budget est voté au niveau du chapitre ou de l'article pour chaque section (fonctionnement et investissement).
String Enumération avec les valeurs suivantes :
  • FcIc : Pour la section de fonctionnement et pour la section d'investissement, le vote est au niveau du chapitre.
  • FaIc : Pour la section de fonctionnement le vote est au niveau de l'article et pour la section d'investissement le vote est au niveau du chapitre.
  • FcIa : Pour la section de fonctionnement le vote est au niveau du chapitre, et pour la section d'investissement, le vote est au niveau de l'article.
  • FaIa : Pour la section de fonctionnement et pour la section d'investissement, le vote est au niveau de l'article.

TYPE TReprRes
Type Description
Cette balise indique la reprise ou non des résultats de l'exercice N-1.
String Enumération avec les valeurs suivantes :
  • 1 : Sans reprise de résultat de l'exercice n-1.
  • 2 : Avec reprise anticipée de résultat de l'exercice n-1.
  • 3 : Avec reprise anticipée de résultat de l'exercice n-1 après le vote du compte administratif de l'exercice n-1.

TYPE TSpecifBudget
Type Description
Cette balise permet de présenter les spécificités budgétaires liées aux EPCI, ASA, AFR…).
1- Présente les spécificités budgétaires liées aux EPCI non dotés d'une fiscalité propre ou aux syndicats mixtes associant des communes et des EPCI.
2- Présente les spécificités budgétaires liées aux ASA AFR.
3- Présente les spécificités budgétaires liées aux SIVU non dotés d'une fiscalité propre.
4- Présente les spécificités budgétaires liées aux SIVU non dotés d'une fiscalité propre.
5- Présente les spécificités budgétaires liées aux budgets annexes d'opération d'aménagement (type ZAC lotissement...).
String Enumération avec les valeurs suivantes :
  • 1 : M14_COM_INF500, M14_COM_500_3500, M14_COM_SUP3500.
  • 2 : M14_COM_INF500.
  • 3 : SIVU doté d'une fiscalité propre.
  • 4 : SIVU non doté d'une fiscalité propre.
  • 5 : M14_COM_INF500, M14_COM_500_3500, M14_COM_SUP3500.

TYPE TTypOpBudg
Type Description
Cette zone permet de déterminer si une opération d'ordre est effectuée à l'intérieur d'une même section ou bien de section à section.
String Enumération avec les valeurs suivantes :
  • 1 : Opération à l'intérieur d'une section.
  • 2 : Opération de section à section.

TYPE TTypOpe

TYPE TTypProv
Type Description

TYPE ATCodTypBud
