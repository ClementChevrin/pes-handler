Détail des codes

TYPE TEtatPJComplt
Type Description
Etat de la PJ complémentaire (cas des titres de recette)
String Enumération avec les valeurs suivantes :
  • 01 : PJ complémentaire adjointe à l’ASAP
  • 02 : PJ complémentaire n’a pu être adjointe à l’ASAP (soit PJ trop volumineuse, soit PJ faisant dépasser le nombre de pages maximal d’un ASAP)
  • 03 : PJ complémentaire non ajointe à l’ASAP car deuxième présentation de la facture à l’éditique suite à réception d’un rejet de l’éditique

TYPE TMotifRejetNotification
Type Description
Informations destinées à déterminer les motifs de rejet détectés par le Pilote de Notification par Mail (redevables reconnus ayant opté pour la dématérialisation)
String Enumération avec les valeurs suivantes :
  • 01 : Adresse mail blacklistée
  • 02 : Adresse mail erroné

TYPE TMotifRejetPostalisation
Type Description
Motifs de rejet détectés par la filière éditique de la DGFiP
String Enumération avec les valeurs suivantes :
  • 01 : ERR008 - Différence majeure détectée entre les lignes d'adresse renseignées dans l'index et le bloc adresse
  • 02 : ERR012 - Anomalie dans le bloc adresse destinataire
  • 03 : ERR020 - Erreur de conversion du PDF en AFP
  • 04 : ERR021 - Erreur de recto verso
  • 05 : ERR022 - Non-respect de la zone de silence de la marge de droite
  • 06 : ERR023 - Non-respect de la zone de silence au-dessus du pavé adresse
  • 07 : WAR022 - Non-respect de la zone de silence autour du CODE 2D
  • 08 : Autre

TYPE TTypBord
Type Description
Type de bordereau.
String Enumération avec les valeurs suivantes :
  • 01 : Ordinaire
  • 02 : Annulation-réduction
  • 07 : Ordres de recette multi créanciers_Reversement BC bénéficiaire

TYPE TTypPce
Type Description
Type de pièce
String Enumération avec les valeurs suivantes :
  • 01 : Titre ordinaire
  • 02 : Titre correctif
  • 10 : Titre en plusieurs années
  • 15 : Titre ORMC (Ordre de recette multi créanciers)
