Détail des codes

TYPE TCCAGReference
Type Description
Référence de CCAG (Cahier des Clauses Administratives Générales) pour un marché public.
String Enumération avec les valeurs suivantes :
  • 01 : T  : CCAG Travaux
  • 02 : FCS : CCAG Fournitures courantes et prestations de Services
  • 03 : MI  : CCAG Marchés publics Industriels
  • 04 : PI  : CCAG Prestations intellectuelles
  • 05 : TIC  : CCAG Techniques de l'information et de la communication
  • 06 : A  : Aucun CCAG visé
  • 07 : MOE : CCAG Maîtrise d'oeuvre
