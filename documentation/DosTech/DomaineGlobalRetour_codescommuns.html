TYPE TCatTiers
Type Description
Catégorie du Tiers.
String Enumération avec les valeurs suivantes :
  • 01 : Personnes physiques.
  • 02 : Personnes morales de droit privé.
  • 03 : Personnes morales de droit public.

TYPE TNatJur
Type Description

  • 01 : Particuliers.
  • 02 : Artisan / Commerçant / Agriculteur.
  • 03 : Société.
  • 04 : CAM ou caisse appliquant les mêmes règles.
  • 05 : Caisse complémentaire.
  • 06 : Association.
  • 07 : Etat ou organisme d'Etat.
  • 08 : Etablissement public national.
  • 09 : Collectivité territoriale / EPL / EPS.
  • 10 : Etat étranger / ambassade.
  • 11 : CAF.
