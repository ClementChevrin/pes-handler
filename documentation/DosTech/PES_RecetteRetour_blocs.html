TYPE TTiers
Nom zone Card(1) Type Description
Bloc Tiers.
InfoTiers 1..1 TInfoTiersRecette Les éléments renvoyés dans le cadre du PESRetour Recette sont :
- Les éléments du référentiel Tiers Hélios dans le cas où la pièce a été rattachée à un Tiers Hélios. Il s'agit des informations du Tiers sur lequel la pièce a été rattachée s'il est isolé, sinon du tiers de référence de son groupe de consolidation.
Dans ce cas, les balises NatIdTiers et IdTiers ne sont pas servies si elles représentent un NIR (NatIdTiers = 04 valeur HELIOS = 4 NUMERO_SS).
- Les éléments du Tiers de la pièce ( tiers de l'encaissement) pour les pièces créées par le comptable (pièces à régulariser). Dans ce cas, les balises NatIdTiers et IdTiers ne sont pas servies si elles représentent un NIR (NatIdTiers = 04 valeur HELIOS = 4 NUMERO_SS).
- Les éléments du Tiers de la pièce (débiteur), pour les pièces rejetées ou mise en instance (dans ce cas en effet, aucun rattachement n'est effectué).
Adresse 0..1 TAdresse Adresse du tiers.
Si l'adresse principale du tiers a pour origine La Poste via les référentiels DGFIP, alors le bloc adresse n'est pas renseigné.
CpteBancaire 0..1 TCpteBancaire Compte bancaire du tiers.

TYPE TBlocPiece
Nom zone Card(1) Type Description
CodServ 0..1 Base_Texte10 Code d'identification du service de l'ordonnateur.
IdPce 1..1 Base_Num13 Identifiant de la pièce. Pour le contexte « Type de bordereau = 05 _ P503 » ou Type de Bordereau = 04_ Bordereau de régularisation » cette balise renvoie l'identifiant de l' encaissement avant émission de titre.
TypPce 1..1 TTypPce Type de pièce. Pour le contexte « Type de bordereau = 05_P503 » ou « Type de bordereau = 04_Bordereau de régularisation » cette balise renvoie au type de pièce 13- Demande émission de titre

Note : dans le cas d'un TypPiece = 13 dans un flux retour, la valeur de la balise IdPiece transmise devra être reprise dans le flux ALLER de régularisation émis par l'ordonnateur ( balise IdPceOrig)
NatPce 1..1 TNatPce Nature de la pièce.

DtePcePec 0..1 Base_Date Date de prise en charge de la pièce.
Remarque :
- Un titre vérifiant DtePcePec et DtePceRej renseignées est un titre qui a été rejeté suite à annulation de prise en charge dans Hélios ; dans ce cas, le titre ne pourra pas être ré-émis par l'ordonnateur avec le même identifiant de pièce (IdPce).
- Un titre vérifiant DtePcePec non renseignée et DtePceRej renseignée est un titre qui a été rejeté en Visa Recette depuis la bannette dans Hélios ; dans ce cas, après correction par l'ordonnateur le titre pourra être ré-émis par l'ordonnateur avec le même identifiant de pièce (IdPce).

ObsEtat 0..1 Base_Texte100 Observation sur l'état de la pièce : codification des motifs. Possibilité de transmettre une pluralité de codification, avec le « - » tiret comme caractère séparateur.
Commentaire 0..1 Base_Texte400 Commentaire saisi par le Comptable lors du visa.
DtePceRej 0..1 Base_Date Date du rejet de la pièce de recette.
